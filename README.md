## Đối với code này thì phải chỉnh một vài chỗ, để có thể tìm kiếm tên của các nhận vật
* Cụ thể hơn thì là đổi link dẫn đến file rimi.csv
* Nếu như không biết chỉnh thì các bạn có thể liên hệ trực tiếp mình :3
## Làm sao để chạy code?
* Đối với code này thì nó chạy trên các nền tảng như DevC++, Codeblock, CP Editor, vscode, ... (không chạy được khi bạn dùng bản web)
* Bạn copy phần hint của character và paste vào phần input của code và chạy code
* Code sẽ giúp bạn tìm kiếm nhanh tên nhân vật và tiện thể thêm luôn ".claim" vào cho bạn
* Nhiệm vụ của bạn chỉ là paste nó vào discord và nhận cards
# Chú ý:
* Không claim quá nhanh (sau một 2 giây mới claim, tính từ lúc nó sinh ra card)
* Không claim quá liên tục (không tham lam mà nhảy giữa nhiều server để lấy card)
## Hiện tại BOT này đang là một con bot nhỏ, nên là nó thay đổi thường xuyên
* Lệnh xem tất cả các tên nhân vật lại bị block, nên mình chỉ có một vài con nhân vật
* Nếu còn thiếu nhân vật nào thì mong các bạn thông cảm và có thể gửi tên nhân vật đó cho mình qua discord
